# 记忆系统管理

> 会话与记忆存储的完整管理

---

## 一、模块定位

记忆系统管理负责智能体的短期记忆、长期记忆和会话管理。

### 核心职责

```
记忆系统管理
├── 会话管理
│   ├── 会话创建/删除/查询
│   ├── 会话状态管理
│   ├── 会话持久化
│   └── 会话恢复
│
├── 短期记忆管理
│   ├── 上下文存储
│   ├── 临时状态管理
│   ├── 记忆窗口控制
│   └── 自动清理
│
├── 长期记忆管理
│   ├── 经验存储
│   ├── 知识存储
│   ├── 向量索引
│   └── 记忆检索
│
└── 记忆统计
    ├── 存储统计
    ├── 使用分析
    ├── 清理策略
    └── 导入导出
```

### 记忆类型

| 类型 | 说明 | 存储位置 | 保留时间 |
|------|------|----------|----------|
| **短期记忆** | 会话上下文、临时状态 | 内存 | 会话结束 |
| **长期记忆** | 经验积累、知识存储 | 向量数据库 | 永久 |
| **会话记忆** | 多轮对话历史 | 内存+持久化 | 可配置 |

## 三、API 端点

### 3.1 会话管理

```python
# 获取会话列表
GET /api/core/memory/sessions
Query: ?status=active&limit=100&offset=0
Response: {
  "sessions": [...],
  "total": 128,
  "active": 45,
  "idle": 30,
  "ended": 53
}

# 创建会话
POST /api/core/memory/sessions
Body: {
  "agent_id": "agent-001",
  "user_id": "user-001",
  "type": "short_term",
  "config": {
    "max_history": 100,
    "recall_count": 5
  }
}
Response: {
  "session_id": "sess-001",
  "status": "active"
}

# 获取会话详情
GET /api/core/memory/sessions/{session_id}
Response: {
  "id": "sess-001",
  "agent_id": "agent-001",
  "user_id": "user-001",
  "status": "active",
  "type": "short_term",
  "created_at": "2026-04-13T10:30:00Z",
  "last_activity": "2026-04-13T10:35:00Z",
  "message_count": 45,
  "memory_size_mb": 2.3
}

# 删除会话
DELETE /api/core/memory/sessions/{session_id}

# 获取会话上下文
GET /api/core/memory/sessions/{session_id}/context
Query: ?limit=100&offset=0
Response: {
  "messages": [...],
  "total": 45
}

# 添加消息到会话
POST /api/core/memory/sessions/{session_id}/messages
Body: {
  "role": "user",
  "content": "查询今日天气"
}
```

## 四、数据模型

```typescript
// 会话定义
interface Session {
  id: string;
  agent_id: string;
  user_id: string;
  status: 'active' | 'idle' | 'ended';
  type: 'short_term' | 'long_term' | 'session';
  config: SessionConfig;
  created_at: string;
  last_activity: string;
  message_count: number;
  memory_size_mb: number;
}

// 会话配置
interface SessionConfig {
  max_history: number;
  recall_count: number;
  auto_cleanup: boolean;
  cleanup_days?: number;
}

// 消息定义
interface Message {
  id: string;
  session_id: string;
  role: 'user' | 'agent' | 'system';
  content: string;
  metadata?: {
    tool_calls?: ToolCall[];
    tokens?: number;
    duration_ms?: number;
  };
  created_at: string;
}

// 工具调用
interface ToolCall {
  tool: string;
  input: any;
  output: any;
  duration_ms: number;
}

// 记忆统计
interface MemoryStats {
  total_memories: number;
  short_term: number;
  long_term: number;
  session: number;
  storage_size_gb: number;
  today_queries: number;
}

// 记忆检索结果
interface MemorySearchResult {
  content: string;
  score: number;
  metadata: {
    session_id?: string;
    created_at: string;
    type: string;
  };
}
```
