# 适配器管理

> LLM适配器配置与监控的完整管理

---

## 一、模块定位

适配器管理负责 LLM 适配器的配置、API 密钥管理、模型参数配置和调用监控。

### 核心职责

```
适配器管理
├── 适配器配置管理
│   ├── 适配器添加/删除
│   ├── API密钥配置
│   ├── 模型参数配置
│   └── 适配器启用/禁用
│
├── 模型管理
│   ├── 模型列表
│   ├── 模型参数
│   ├── 模型限流
│   └── 模型测试
│
├── 连接管理
│   ├── 连接测试
│   ├── 连接状态监控
│   └── 连接池管理
│
└── 调用监控
    ├── 调用统计
    ├── 错误监控
    ├── 耗时分析
    └── 成本追踪
```

### 支持的适配器

| 适配器 | 说明 | 模型 |
|--------|------|------|
| **OpenAI** | OpenAI API | GPT-4, GPT-3.5, Embeddings |
| **Anthropic** | Claude API | Claude-3, Claude-2 |
| **Azure OpenAI** | Azure OpenAI | GPT-4, GPT-3.5 |
| **本地模型** | 本地部署 | Ollama, vLLM, LLaMA |

### 3.1 适配器管理

```python
# 获取适配器列表
GET /api/core/adapters
Query: ?provider=openai&status=active&limit=100&offset=0
Response: {
  "adapters": [
    {
      "id": "adapter-001",
      "name": "openai-1",
      "provider": "OpenAI",
      "status": "active",
      "today_calls": 25000,
      "success_rate": 0.992
    }
  ],
  "total": 4
}

# 添加适配器
POST /api/core/adapters
Body: {
  "name": "openai-1",
  "provider": "OpenAI",
  "description": "主要使用的OpenAI适配器",
  "api_key": "sk-xxx",
  "api_base_url": "https://api.openai.com/v1",
  "organization_id": "org-xxx",
  "models": [
    {"name": "gpt-4", "enabled": true, "max_tokens": 8192, "temperature": 0.7}
  ],
  "rate_limit": {
    "rpm": 60,
    "tpm": 90000,
    "daily_quota": 1000000
  },
  "retry": {
    "max_retries": 3,
    "interval_seconds": 1,
    "backoff": "exponential"
  }
}
Response: {
  "id": "adapter-001",
  "status": "created"
}

# 获取适配器详情
GET /api/core/adapters/{adapter_id}

# 更新适配器配置
PUT /api/core/adapters/{adapter_id}
Body: {
  "rate_limit": {
    "rpm": 100
  }
}

# 删除适配器
DELETE /api/core/adapters/{adapter_id}

# 测试连接
POST /api/core/adapters/{adapter_id}/test
Response: {
  "success": true,
  "latency_ms": 150,
  "message": "Connection successful"
}

# 启用/禁用适配器
POST /api/core/adapters/{adapter_id}/enable
POST /api/core/adapters/{adapter_id}/disable
```

## 四、数据模型

```typescript
// 适配器定义
interface Adapter {
  id: string;
  name: string;
  provider: 'OpenAI' | 'Anthropic' | 'AzureOpenAI' | 'Ollama' | 'vLLM' | 'Custom';
  description: string;
  status: 'active' | 'disabled' | 'error';
  api_key: string;
  api_base_url: string;
  organization_id?: string;
  models: ModelConfig[];
  rate_limit: RateLimitConfig;
  retry: RetryConfig;
  created_at: string;
  updated_at: string;
}

// 模型配置
interface ModelConfig {
  name: string;
  enabled: boolean;
  max_tokens: number;
  temperature: number;
  top_p?: number;
  frequency_penalty?: number;
  presence_penalty?: number;
}

// 限流配置
interface RateLimitConfig {
  rpm: number;  // requests per minute
  tpm: number;  // tokens per minute
  daily_quota: number;
  on_limit: 'wait' | 'error' | 'fallback';
}

// 重试配置
interface RetryConfig {
  enabled: boolean;
  max_retries: number;
  interval_seconds: number;
  backoff: 'linear' | 'exponential' | 'constant';
}

// 调用统计
interface CallStats {
  total_calls: number;
  success_rate: number;
  avg_duration_ms: number;
  tokens_used: number;
  errors: Record<string, number>;
}

// 调用记录
interface CallRecord {
  id: string;
  adapter_id: string;
  model: string;
  status: 'success' | 'error';
  duration_ms: number;
  tokens: {
    prompt: number;
    completion: number;
  };
  error?: string;
  timestamp: string;
}
```
